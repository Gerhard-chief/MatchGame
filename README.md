# MatchGame (WPF)

A small learning project from *Head First C#* — a simple “match pairs” game built with WPF.

## Features
- Timer (tenths of a second)
- Match-pair logic
- Game reset / new shuffle

## Tech Stack
- C#
- .NET
- WPF (Desktop)

## How to Run
1. Open `MatchGame.sln` in Visual Studio
2. Restore NuGet packages (if prompted)
3. Press **F5** to run

## Project Status
Work in progress. This is a learning repo, so commits may be frequent and not always “clean”.
